## 🚀 Как Использовать

### Для Пользователя:
1. Откройте любой аспект
2. Перейдите на вкладку **"Практики"**
3. Выберите 1-2 упражнения и начните практиковать
4. Изучите **"Интеграцию с Противоположностью"**
5. Перейдите на вкладку **"Синергия"**
6. Обратите внимание на **"Соматические Маркеры"**

### Для Разработчика:
```bash
# Установка
npm install

# Запуск
npm run dev

# Сборка
npm run build

# Деплой
npm run deploy
```
