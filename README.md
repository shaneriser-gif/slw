# Соционика - Колесо Баланса

Веб-приложение для самооценки и отслеживания развития по 8 аспектам соционики.

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для production
npm run build

# Предпросмотр production сборки
npm run preview
```

## Деплой на GitHub Pages

```bash
# Деплой
npm run deploy
```

Перед деплоем убедитесь, что в `vite.config.js` указан правильный `base` (название вашего репозитория).

## 🎯 Как Использовать

1. **Оцените аспекты** — поставьте оценку от 1 до 10 каждому аспекту
2. **Изучите теорию** — узнайте о сути, дилеммах и архетипах
3. **Пройдите самооценку** — ответьте на вопросы по микрополям
4. **Выберите практики** — начните с 1-2 упражнений
5. **Изучите синергию** — поймите связи между аспектами
6. **Ставьте цели** — определите направление развития
7. **Ведите дневник** — отслеживайте прогресс и инсайты
