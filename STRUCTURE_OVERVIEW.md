## 🔗 Связи Между Аспектами

```
        ЧЛ ←→ БЭ
       ↗  ↘   ↗  ↘
     БЛ    ЧЭ    БС
       ↖  ↗   ↖  ↗
        ЧС ←→ БИ
         ↕
        ЧИ

Интеграция (противоположности):
  ЧЛ ↔ БЭ
  БЛ ↔ ЧЭ
  ЧС ↔ БИ
  БС ↔ ЧИ

Синергии (примеры):
  ЧЛ + ЧИ = Инновации
  БЭ + БС = Забота и Комфорт
  ЧС + ЧЭ = Харизматичное Лидерство
  БЛ + БИ = Философская Система
```

## 📱 Пользовательский Путь

```
1. Пользователь оценивает аспекты (1-10)
   └── Видит Колесо Баланса

2. Выбирает аспект для развития
   └── Читает Теорию

3. Проходит Самооценку
   └── Отвечает на вопросы по микрополям

4. ✨ Переходит в Практики
   └── Выбирает 1-2 упражнения
   └── Изучает Интеграцию с противоположностью

5. ✨ Изучает Синергию
   └── Понимает связи с другими аспектами
   └── Видит соматические проявления

6. Ставит Цели
   └── Отвечает на вопросы для целей

7. Ведёт Дневник
   └── Записывает инсайты и прогресс
```
